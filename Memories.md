# Boutique Rumble: Architectural Memory

## Project Ethos
- **Paradigm**: Boutique (Curated constraints over parameter density).
- **Core Aesthetic**: Rhythmic "Rumble" bass (Skrillex/Fred Again style).
- **Primary Interface**: The "Boutique 5" Macros.

- **[2026-04-18] Hardcoded 20% Random Step-Skipping for rhythmic syncopation**: On each gate-phase wrap, `RumbleEngine` rolls `entropyRandom` (~20% chance) to set `mCurrentStepIsSkipped`; while set, `getGateTarget()` returns silence for that rhythmic cycle so roughly one in five pulses drops out for syncopated rumble.
- **[2026-04-18] Added dynamic SKIP slider (0-100%) for real-time rhythmic chaos control**: APVTS float `skip_prob` / UI label `SKIP` (default 0.2) drives `RumbleEngine::setSkipProbability`; at 0 the gate never skips, at 1 every wrapped step is skipped (`nextFloat() < 1`) for full rhythmic silence through the pulse.
- **[2026-04-18] Implemented Isolated Sub-Drift (3:7:11) mapped to the GIRTH knob**: Three free-running sine LFOs at 0.31 / 0.73 / 1.13 Hz (prime-scaled “3:7:11” character) sum to a normalized depth; `GIRTH` scales fine detune of Mid A/B up to ±20 cents via `2^(cents/1200)`, while mid oscillators keep advancing phase when inactive so short gates sample evolving drift.
- **[2026-04-18] Implemented Legato frequency smoothing and Gate-Dip re-triggering to eliminate note-change clicks**: `noteOn` only uses `LinearSmoothedValue::setTargetValue` for pitch when a voice is already active; legato stacked notes trigger a ~1 ms gate-target dip to zero (via existing `gateSlew` state, no `reset`) before the LFO target returns; `PULSE` slew times are clamped to at least 1 ms so square edges never use a true 0 ms transition.
- **[2026-04-18] Implemented 25Hz Safety Rail and DC Blocker to preserve infrasonic character while protecting hardware**: After GRIT/spatial summing and note envelope, each channel runs a 5 Hz first-order DC blocker then a 4th-order Linkwitz–Riley high-pass at 25 Hz so sub-audible fundamentals (e.g. C-2 ~4 Hz) are attenuated while harmonics and motion from upstream saturation remain audible.
- **[2026-04-18] Adjusted release curves to 100ms exponential to eliminate infrasonic release pops**: Note envelope release is exponential (`envelope *= exp(-1/(fs·τ))`) with ~35 ms base τ scaled ~±55% by MIDI note (longer τ for low notes); attack stays ~5 ms linear. The 25 Hz safety high-pass runs before the 5 Hz DC blocker so the DC stage is last and catches release thumps.
- **[2026-04-18] Implemented 45ms exponential Pitch-Thump (+36st) for percussive note-on transients**: Fresh `noteOn` seeds a +36 semitone offset that decays multiplicatively to ~0 in exactly 45 ms (matched to sample rate); legato retriggers cap at +18 st. Effective base Hz is `f * 2^(thump/12)` before Sub-Drift, so sub and mids stay phase-aligned through the stack.
- **[2026-04-18] Implemented comprehensive BRAKE slider with quadratic scaling for manual Tape-Stop performance**: APVTS float `BRAKE` (default 1.0) feeds a 50 ms `LinearSmoothedValue`; `brakeRaw²` scales oscillator frequencies and post-envelope / post-gate level before the safety rail for coordinated pitch and level “tape stop” braking.
- **[2026-04-18] Transformed BRAKE slider into a multi-stage Stall Manifold with rate-modulated stuttering**: Below ~0.35 smoothed brake, a square stutter gate (phase accumulator) multiplies the brake multiplier; stutter rate ramps 25 Hz→3 Hz toward stall, duty shrinks for choke; at or below ~0.05 output dies; above the stall threshold behavior is smooth quadratic braking only.
- **[2026-04-18] Redesigned BRAKE as a Mechanical Stress macro with gain-floor protection and rate-modulated gasping stutters**: Smoothed `brakeIn` drives cubic **pitch** pull, `jmap` **gain** (20% floor to full); below 0.6 a free-running square stutter (30 Hz→2 Hz, 50%→5% duty) **replaces** the PULSE/LFO gate while transport phase still advances; below 0.05 full silence.
- **[2026-04-18] Recalibrated BRAKE slider with sqrt-warping and a 20Hz frequency floor for expanded performance resolution**: `warpedBrake = sqrt(brakeIn)` drives cubic pitch and stutter maps; sub frequency is clamped to ≥20 Hz while `brakeIn ≥ 0.1`; stutter engages when `warpedBrake < 0.9` with rate 30→2 Hz and linear `gainMult = brakeIn` (full cut only below the hard stop threshold).
- **[2026-04-18] Implemented Sigmoid input warping and Logarithmic stutter rating for a non-linear, boutique BRAKE feel**: `warpedBrake = 1/(1+exp(-(brakeIn-0.5)*10))` for cubic pitch; stutter depth fades in from raw `0.8→0.6`; below `0.6` stutter rate follows log interpolation **4 Hz→30 Hz** toward the hard stop, with **20 Hz** sub floor for `brakeIn ≥ 0.05`.
- **[2026-04-18] Replaced hard-threshold stutter with a continuous power-curve (1-B)^3 rate and fractional intensity mapping**: `stutterIntensity = 1 - brakeIn`, `stutterRate = 40·(1-brakeIn)³` (0 Hz at full brake, ~0.32 Hz at 0.8, ~40 Hz near zero); smooth gate `1 - intensity·(1 - rawSquare)` multiplies the PULSE gate; sigmoid cubic pitch, linear gain, **20 Hz** sub floor while `brakeIn > 0`.
- **[2026-04-18] Reduced attack pitch thump from +36st to +18st for a more mechanical, less digital impact**: changed `thumpSemitoneFresh` to `18.0f` while keeping the existing fast thump decay timing untouched.
- **[2026-04-19] Implemented Adaptive Release Scaling to prevent pulse-overlap and maintain sub-sonic tightness**: `RumbleEngine` derives `pulsePeriodMs` from BPM × subdivision and caps the exponential note-off τ using the lesser of ~80% of that period and (period − 5 ms) hard silence; the coefficient refreshes each block (and on rate/BPM changes) so fast pulse rates stay punchy instead of smearing into a drone—**independent of GIRTH**, which only shapes the gate blend and spatial/drift behavior.
- **[2026-04-19] Verified LFO/Envelope decoupling to ensure Girth expansion does not affect temporal periodicity**: Renamed the free-running accumulator to `mPulsePhase` (BPM × subdivision only), removed Girth from adaptive release math, locked the first few phase samples of each cycle to the square onset so morph timing cannot “pull” the downbeat, retrigger non-legato `noteOn` from a silent envelope to cut release tails, and added throttled `DBG` logging of `timeBetweenOnsets` in Debug builds to confirm onset spacing is identical at Girth 0 vs 1.
- **[2026-04-19] Implemented Note-Specific Seeded Motif Engine on SKIP slider for deterministic, repeating rhythmic signatures**: Replaced per-pulse random skipping with a 16-bool `mMotifPattern` from `std::mt19937` seeded by `note + (int)(SKIP*100000)`; SKIP sets Bernoulli density (~SKIP fraction of silent steps); the pattern advances each LFO wrap and realigns to step 0 on host bar boundaries (`floor(ppq/4)`); SKIP `0` bypasses to all-true; the motif is rebuilt on non-legato or note-change `noteOn` and when SKIP moves during sustain.
